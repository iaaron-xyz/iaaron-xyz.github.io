## Search a font

You can find fonts on sites like google fonts, once you find it, you can download it. After that unzip the fonts if necessary and place them in the public folder of your project: `project-folder/public`

2. Download the font and place it in your folder project.
3. Create a .css for that file.
4. Add a font name to the tailwindcss file.
5. Call the css file into your main css file.
