Step 1: Install NVM

What is NVM?

NVM stands for Node Version Manager, is a shell script that helps in the management of multiple Node.JS versions on a per-user basis. Using NVM, you can install, uninstall and set a specific Node.JS version as the default version.

The repository and the most updated url can be found in their Github NVM Page. Instructions followed also from this guide.

How to install and keep udated NVM

The same command can be used to install and to update NVM changing its version to the latest one. Yo can install it the next way, based on their Github NVM Page. 

                ➜  ~ curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.1/install.sh | bash
